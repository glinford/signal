# signal
Signal Lib

### Build in Javascript

## Dependencies :
NPM > 3.3.0
Node.js > 5.7.0

## Installation :
```shell
npm install
```

### Usage Exemples :
``` shell
node do.js 123 0 1221 4353 22 113
one hundred and twenty-three
zero
1221 not allowed
4353 not allowed
twenty-two
one hundred and thirteen
```

``` shell
node do.js 10 -20 -1.23 3.87 john12 89kiss hello
```
